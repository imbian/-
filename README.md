参考资料  
1.  [国学导航－中国历史地图集](http://www.guoxue123.com/other/map/zgmap/index.htm)  
为纸质历史地图的扫描版本，并且使用的是谭其骧主编的中国历史地图集，只有一级行政区的地图。  
2. [观沧海](https://www.ageeye.cn/)  
一个志愿者在线协作项目，宏大而进展缓慢，但是它有一个优点，是矢量地图，可以放大缩小。  
3. [osgeo中国历史地图](https://www.osgeo.cn/list/gzgls)  
也是扫描版本，清晰度一般。资料和索引比较全，查找方便。
